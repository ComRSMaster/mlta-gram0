# Интерпретатор грамматик класса 0

Математическая логика 2026, группа Б05-515, Хохлов Владислав

## Описание

**Грамматика класса 0** - набор из $\Sigma$, $\Gamma$, $P$, $S$, где:

* **Терминалы ($\Sigma$)** - символы, из которых состоят итоговые слова.
* **Нетерминалы ($\Gamma$)** - технические символы, которые нужны для вспомогательные для обработки данных.
* **Правила ($P$)** - список замен вида $\alpha \to \beta$: подстрока $\alpha$ заменяется на строку $\beta$. В $\alpha$ должен быть хотя бы один нетерминал.
* **Начальный символ ($S$)** - программа начинает применять правила ($P$) к строке из одного нетерминального символа $S$.

**Вывод** - последовательность строк, полученных из $S$ последовательным применением правил $\alpha \to \beta$. Вывод считается завершенным.

Программа завершается и вывод считается завершенным, когда в строке отсутствуют технические нетерминальные символы.

## Синтаксис

Каждый **символ** в грамматике - строка `s`, состоящая из строчных и заглавых латинских букв, цифр и символов `` !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~ ``, символ **не может начинаться с решетки (`#`)**, также символ **не может быть стрелкой (`->`)**.

**Строка** - последовательность этих символов, разделенных пробелом. Можно задать свой разделитель для вывода символов в консоль через флаг `-s` (по умолчанию разделитель - пробел).

**Нетерминальные символы ($\Sigma$)** - символы, которые начинаются с точки (`.s`), в частности символ, состоящий только из точки тоже нетерминальный.

**Терминальные символы ($\Gamma$)** - символы, которые начинаются не с точки (просто `s` без точки в начале).

**Начальный символ ($S$)** - просто точка (`.`).

В файле каждая строка либо пустая, либо комментарий (должен стоять символ решетки, а после него может идти любая последовательность символов), либо само правило $\alpha \to \beta$ - заменить подстроку `` a₀ a₁ a₂ ... aₙ `` на `` b₀ b₁ b₂ ... bₘ ``, пробелы отделяют разные символы и зарезервированный символ стрелки:
```
a₀ a₁ a₂ ... aₙ -> b₀ b₁ b₂ ... bₘ
```
